# SiegePrep

## 🚀 Live Demo

[Open SiegePrep](https://siege-prep.vercel.app)

**SiegePrep** is a minimalist, strategy-inspired study preparation tool that visualizes your syllabus as an active battlefield.

Instead of tracking progress with boring checklists, SiegePrep forces you to confront your preparation like a tactical problem — where every topic is an incoming threat, and your goal is to neutralize them before the exam.

---

## 🧠 Concept

- A central **fortress** represents your preparedness.
- Each **topic becomes an enemy unit** approaching it.
- Difficulty determines how close the threat is:
  - Easy → far away
  - Very Hard → dangerously close

Progress isn’t cosmetic — it reflects actual effort and verification.

---

## ⚙️ How It Works

Each topic moves through strict states:

- **Unseen** → not started
- **Engaged** → you began studying
- **Damaged** → partial understanding
- **Defeated** → verified mastery

You cannot skip steps or fake progress.

---

## 🎮 Core Mechanics

- **Start Study** → engage a topic
- **Mark Study** → partial progress
- **Tested** → confirm mastery (self-verified)
- **Retreat** → admit failure (with penalty)
- **Delete** → remove irrelevant topics

---

## ❤️ Player HP System

Your preparation has consequences:

- Retreat → -10 HP
- Failed test → -15 HP
- (Optional) Completion reward → +5 HP

If HP drops too low:

> ⚠️ _You are underprepared._

---

## 🛰️ Battlefield Visualization

- Topics are rendered as **unit shapes**:
  - Troops, Tanks, Bombers, Ships
- All units **face the fortress**
- A rotating **radar sweep** adds situational awareness
- No clutter — only essential information is visible

---

## 📊 Test Management

- Create multiple test scenarios
- Input topics manually with difficulty
- Progress is saved using `localStorage`
- Switch or delete tests anytime

---

## 🎯 Victory Condition

When all topics are defeated:

> ✅ **ALL TARGETS ELIMINATED**

You’re ready.

---

## 🛠️ Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- No frameworks, no dependencies

Runs directly in the browser.

---

## 🚫 What This Is NOT

- Not a gamified distraction
- Not a productivity dashboard
- Not a passive tracker

This is a **pressure tool** — it reflects how prepared you actually are.

---

## 📌 Why SiegePrep?

Most study tools reward effort.  
SiegePrep forces **accountability**.

You either defeat the syllabus — or it overwhelms you.
